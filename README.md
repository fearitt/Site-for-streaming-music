# Site-for-streaming-music

Стриминговый сервис с админ-панелью, загрузкой треков и импортом с YouTube.

## Возможности

- **Два режима**: Администратор (добавление/удаление треков) и Пользователь (только добавление)
- **Загрузка файлов**: mp3, wav, flac, ogg, aac, m4a, wma
- **Импорт по ссылке**: YouTube, SoundCloud и другие сайты через yt-dlp
- **Встроенный плеер**: play/pause, prev/next, прогресс-бар, регулировка громкости, автопереключение

## Установка и запуск

```powershell
git clone https://github.com/fearitt/Site-for-streaming-music.git
cd Site-for-streaming-music
npm install
npm start
